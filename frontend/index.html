METHODOLOGY

RL Architecture

How the Q-Learning agent decides where to deploy firefighting resources.

🌍

Environment

10×10 grid wildfire simulator with probabilistic fire spread and wind bias

→
👁️

State

Binned (burning, trees) per quadrant
(b₁,t₁, b₂,t₂, b₃,t₃, b₄,t₄)

→
🤖

Q-Learning Agent

ε-greedy action selection
Q(s,a) += α[r + γ·max Q(s',a') - Q(s,a)]

→
🎯

Action

Select sector (0–3) to deploy firefighting resources

→
⭐

Reward

−(burned) + 2.0 × (suppressed)
Penalise fire, reward action

Hyperparameters

α (Learning Rate) 0.1
γ (Discount Factor) 0.95
ε (Start → Min) 1.0 → 0.05
ε Decay 0.995
Grid Size 10 × 10
Max Steps 50
Sectors 4 (2×2)
State Space ~6561
